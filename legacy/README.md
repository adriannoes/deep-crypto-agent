# Legacy Files

This directory contains legacy files from the original ABU quantitative trading system.

## Contents

### notebooks/

Contains legacy Jupyter notebooks from the original ABU system. These notebooks are in Chinese and contain examples, tutorials, and documentation from the original implementation.

**Note:** These files are kept for historical reference only. The modern implementation is in the `crypto_quant_pro/` directory.
