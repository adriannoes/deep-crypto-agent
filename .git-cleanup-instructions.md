# Instruções para Limpeza do Git

Este arquivo contém instruções para limpar o status do git conforme planejado.

## Comandos a Executar

```bash
# 1. Adicionar arquivos deletados ao stage
git add -u

# 2. Adicionar diretório legacy/ ao controle de versão
git add legacy/

# 3. Commitar as mudanças
git commit -m "chore: organizar estrutura do projeto e adicionar legacy/"

# 4. Verificar status
git status
```

## Arquivos que serão commitados

- Arquivos deletados (notebooks antigos movidos para legacy/)
- Diretório legacy/ completo
- Todas as novas configurações e documentação criadas

## Nota

Execute estes comandos manualmente quando estiver pronto para commitar as mudanças.
